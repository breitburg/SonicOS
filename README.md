# SonicOS

HomeKit-enabled ESP32-firmware built with HomeSpan for "Sonic Neon Sign"

## Built with

- HomeSpan
- Arduino Core

## Build

Go to Arduino IDE, install all dependencies, connect ESP32 and push "Upload" button.
